Feature project: shopping website building from IAT339

Cocomolo shopping website

We used html, css, and javascript, and unlike our previous group work, we were asked to use git to build the web pages and observe our changes and progress on git. I personally think that git is a good platform, not only can it be open source, and it is very easy for my clients to view my website. The design and discussion of style sheets is especially critical when doing basic programming. I am very passionate about consistency in web design and the importance of improving the user experience.

Reflection:

In this project we were given a limited amount of time to complete the project, and if we had more time, we would have set up more places on the page to utilise js in order to make the process smoother and smoother for the customer.

First step

Initializing

The project started with a overall styleguide and both of teammates sharing each other's idea. We finally decided the brand: COCOMOLO.

Second step

Developing and building

Cocomolo is a website designed to promote and sell handmade decorative items, and it is an attempt by the Cocomolo brand to operate on the Internet. At the same time, it uses a minimalist style to show customers different products, and customers can browse different pages with detailed information and reviews of different products. Besides, it is a responsive website, what is a responsive website? Customers can also browse our website on their cell phones and also do the basic process of shopping on their cell phones.
